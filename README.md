# DPG-RPS
## *Rock, Paper, Sockets!*


### Prequisites
* poetry


### Installation
1. `git clone https://github.com/Jah-On/DPG-RPS.git`
1. `cd DPG-RPS/`
1. `poetry install`
1. `poetry run python src/main.py`

### Build the docs
1. `poetry run doit makedocs`

HTML files are outputted to `docs/src`
